Crypto Tracker (Windows EXE)

Crypto Tracker is a standalone Windows desktop application that tracks live cryptocurrency prices in CAD using CoinGecko.

Features

Live CAD crypto prices

User-defined price alerts (≥ / ≤)

24-hour low/high potential & profit estimates

Optional email alerts using your own email settings

No server, no accounts, no tracking

How to use

Download the EXE

Double-click to run

Add coins and set alerts

(Optional) Enter your own email settings for alerts

Requirements

Windows 10 or newer

Internet connection for price updates

Privacy & safety

Runs entirely on your computer

No data is sent anywhere except:

CoinGecko (price data)

Your email provider (only if you enable email alerts)
